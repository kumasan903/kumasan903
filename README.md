# 今までに作ったもの
## フライトシミュレーター用降下計算ツール
飛行機は降下するのに時間も距離も必要。  
高高度での巡航(水平飛行)から、目標高度までの降下に  
どれくらいの距離が必要かを求めるシンプルな計算機。  
最初はPython+tkinterでデスクトップアプリとして作っていたが後にJSに移行。
